Hause Price Prediction
Using "Neural-Networks"

Developed deep neural networks with Keras to predict housing prices, handling data preprocessing including imputation, encoding and train-test split. Experimented with network architectures by tuning depth, layers, regularization via dropout and hyperparams like batch size and epochs. Used MSE as loss function, Adam optimizer and metrics like MAE and R-squared to evaluate model performance. Identified optimized topology with 64-unit dense layers and ReLU activation that obtained lowest error and highest accuracy in predicting prices on test data. Demonstrated end-to-end deep learning proficiency through data pipelines, neural network design, training and controlled testing to drive model performance on key industry forecasting task.

Stroke Diagnosis
Using KNN

The goal of this machine learning project was to develop a model for reliably diagnosing breast cancer based on tumor cell measurements. The dataset contained over 500 tumor data points with 30 input features derived from digitized images and a target label indicating malignant or benign diagnosis. After exploring the medical data, I engineered the raw measurements into usable inputs features and split them into train and test sets. Multiple classification models including Logistic Regression, KNN and Random Forest were built using scikit-learn, trained on the data and evaluated on precision, recall and accuracy metrics to compare their performance. Additionally, techniques like grid search cross-validation were employed to tune the hyperparameters of each model. The Random Forest classifier with optimized parameters achieved the highest accuracy of 96% on the test set, demonstrating effective learning from the data patterns. Overall, this end-to-end project involved data preprocessing, applying ML best practices, systematic benchmarking and selection to create an automated breast cancer screening system ready for real clinical usage and impact.
